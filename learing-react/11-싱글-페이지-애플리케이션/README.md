# 핵심 목표

**React Router를 활용한 SPA 라우팅과 UX 최적화**

- **SPA 라우팅의 원리와 환경 구축**  
`react-router-dom`의 핵심 컴포넌트인 `BrowserRouter`와  
`Routes`의 역할을 이해하고, 페이지 단위로 컴포넌트를 구조화하여  
새로고침 없이 전환되는 싱글 페이지 애플리케이션(SPA)의 기초를 확립합니다.
- **동적 라우팅과 데이터 연동**  
`useParams`와 쿼리 스트링(`useSearchParams`)을 활용해 URL 속  
동적 데이터를 추출하고, 이를 기반으로 상세 페이지와 필터링된 목록을  
구현하여 사용자 맞춤형 콘텐츠를 제공하는 능력을 기릅니다.
- **레이아웃 설계와 고급 라우팅 기법**  
`Outlet`을 이용한 중첩 라우팅(Nested Routes)으로  
공통 레이아웃을 효율적으로 관리하며, `ProtectedRoute`와  
`Navigate`를 구현하여 권한에 따른 접근 제어 및  
사용자 흐름(User Flow)을 정교하게 설계합니다.
- **사용자 경험(UX) 고도화 및 브라우저 메커니즘**  
페이지 이동 시 스크롤 최상단 이동 처리와 같은 디테일한 UX 개선 기법을 익히고,  
브라우저 History API의 작동 원리를 이해하여 뒤로 가기/앞으로 가기가  
보장되는 안정적인 내비게이션 환경을 구축합니다.
