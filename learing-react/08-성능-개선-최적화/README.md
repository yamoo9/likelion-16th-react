# 핵심 목표

**리액트 성능 최적화 (Optimization)**

- **렌더링 메커니즘의 이해와 병목 탐색**  
리액트의 리렌더링 조건과 부모-자식 간의 렌더링 전파 원리를 명확히 이해하고,  
브라우저의 프로파일러 도구를 활용해 성능 저하를 일으키는 구체적인  
병목 구간을 스스로 찾아내는 분석 능력을 기릅니다.
- **구조적 설계를 통한 불필요한 렌더링 격리**  
단순히 훅을 사용하는 것을 넘어, `children` props 설정이나  
컴포넌트 계층 구조 변경 등 구조적인 접근을 통해 부모의 상태 변화가  
자식에게 미치는 영향을 최소화하고 참조를 유지하는 최적화 기법을 습득합니다.
- **메모이제이션(Memoization)의 전략적 활용**  
`React.memo`, `useMemo`, `useCallback`의 동작 원리와  
얕은 비교(Shallow Compare) 개념을 파악하여 무거운 연산과   
함수 재생성을 방지하되, "과한 최적화"의 부작용을 경계하며  
실무 상황에 맞는 적절한 최적화 시점을 판단하는 선구안을 확보합니다.
