### 실습 요구사항

**목표: 불변성을 지키는 장바구니(Shopping Cart) CRUD 구현**

1. **데이터 구조**
    `{ id: number, name: string, count: number }` 형태의   
    객체 배열을 상태(`cart`)로 관리하세요.
    
2. **필수 기능 (3가지)**
    - **추가 (Create)**: '상품 추가' 버튼 클릭 시 목록에 새 상품(예: 사과)을 추가하세요.
    - **삭제 (Delete)**: 항목 옆 '삭제' 버튼 클릭 시 해당 상품을 제거하세요.
    - **수정 (Update)**: `+`,  버튼으로 수량을 변경하세요.
3. **제약 사항 (Mutation 금지)**
    - `push()`, `pop()`, `splice()`, `reverse()`, `sort()` 사용 금지.
    - `items[index] = ...` 와 같은 직접 할당 금지.
4. **확인 사항**
    - 수량 변경 시 상품의 이름(`name`)이 사라지지 않는지 확인하세요.
    - 수량이 1 미만으로 내려가지 않도록 처리해 보세요. (선택 사항)
