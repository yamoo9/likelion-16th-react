# 핵심 목표

**리스트 렌더링과 컴포넌트 설계**

- **효율적인 리스트 렌더링과 Key의 본질 이해**  
JavaScript의 `map()` 메서드를 활용해 배열 데이터를 동적인 UI로 변환하는  
패턴을 익히고, 리액트 재조정(Reconciliation) 알고리즘의 핵심인 `key` 속성을  
통해 컴포넌트의 정체성을 유지하고 성능을 최적화하는 원리를 체득합니다.
- **데이터 필터링과 상태 관리의 최소화 원칙**  
`filter()` 메서드를 활용한 검색 및 카테고리 필터링 기능을 구현하며, 원본 데이터는  
보존하고 파생된 상태(Derived State)를 계산하여 사용하는 방식을 통해 불필요한  
`useState` 남발을 억제하는 클린 코드 설계 능력을 기릅니다.
- **컴포넌트 추상화와 조건부 UI 설계**  
반복되는 리스트 아이템을 독립적인 컴포넌트로 분리하고 Props 인터페이스를 정의하여  
재사용성을 높이며, 데이터 로딩·빈 데이터·품절 상태 등 다양한 비즈니스 상황에  
대응하는 선언적 조건부 렌더링 기법을 실무 수준으로 끌어올립니다.
