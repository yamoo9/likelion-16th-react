# 핵심 목표

**DAY 05. useEffect와 비동기 데이터 통신**

- **Side Effect의 제어와 생명주기 이해**  
React 컴포넌트의 순수성을 유지하면서 외부 시스템(API, 타이머, 이벤트)과 상호작용하는  
`useEffect`의 원리를 파악하고, 의존성 배열(`deps`)과 클린업(`Cleanup`) 함수를 통해  
메모리 누수 없는 안전한 Side Effect 관리 능력을 습득합니다.
- **비동기 데이터 페칭과 상태 관리**  
`fetch` API와 `async/await`를 활용해 외부 데이터를 동적으로 불러오며, 로딩(Loading),  
에러(Error), 성공(Success)의 3단계 상태 처리를 통해 사용자에게 끊김 없는 데이터 기반  
UI 경험을 제공하는 패턴을 체득합니다.
- **네트워크 최적화와 경쟁 상태(Race Condition) 해결**  
빠른 연속 입력 상황에서 발생하는 경쟁 상태 문제를 `AbortController`를 이용한  
요청 취소와 `Debounce` 개념으로 해결하고, 실무 수준의 검색 자동완성 기능을  
구현하여 애플리케이션의 성능과 데이터 정합성을 최적화합니다.
