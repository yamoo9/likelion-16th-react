# 이 프로젝트는 당신이 알고 있는 일반적인 Next.js가 아닙니다.

- 이 프로젝트는 **Next.js 16** 및 **Tailwind CSS v4**를 사용합니다.
- 파괴적 변경 사항(Breaking Changes)이 포함되어 있어, API와 파일 구조가 당신의 학습 데이터와 다를 수 있습니다. 

코드를 작성하기 전에 반드시 `node_modules/next/dist/docs/`에 있는 관련 가이드를 읽으십시오. 
지원 중단(Deprecation) 알림에 주의를 기울이십시오.
